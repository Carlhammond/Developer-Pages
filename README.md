## Adding a new page

1. Create a folder named after the app (e.g. `My-App/`).
2. Add an HTML file inside it (e.g. `privacy.html`).
3. The page will be live at `https://<username>.github.io/Developer-Pages/My-App/privacy.html` once pushed.
